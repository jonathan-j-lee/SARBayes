# SARBayes

Forecast lost person survival using wilderness search-and-rescue (WiSAR) data.

## Setup

```bash
cd SARBayes/
sudo pip3 install -r requirements.txt
ln -s data/isrid-snapshot-your-version-here.db data/isrid-master.db
```
